# Do Movie Reviews Actually Say What They Mean? Comparing Sentiment Analysis Models on Letterboxd Reviews

## Research Question

Which sentiment analysis model ‚Äî VADER or HuggingFace RoBERTa ‚Äî more accurately predicts the sentiment implied by Letterboxd star ratings, and what textual features influence model performance?

## Overview

This case study evaluates two sentiment analysis approaches ‚Äî the rule-based VADER model and a transformer-based HuggingFace RoBERTa model ‚Äî on 4,755 Letterboxd movie reviews. Ground-truth labels are derived from star ratings, and the two models are compared on accuracy, Macro-F1, and how each handles the severe class imbalance present in the dataset (~93% positive reviews). Students run VADER locally and compare its outputs to pre-computed HuggingFace predictions.

## Data

**`DATA/letterboxd_reviews.csv`** ‚Äî 4,755 Letterboxd reviews of Top 250 films.

| Column | Description |
|---|---|
| `review_text` | Written review text |
| `star_rating` | 0.5‚Äì5.0 in 0.5 increments |
| `true_label` | Ground truth: `negative` / `neutral` / `positive` |
| `pred_label_thresh` | Pre-computed HuggingFace RoBERTa prediction |
| `vader_label` | VADER prediction (added by `run_analysis.py`) |
| `has_emoji` | Boolean ‚Äî review contains at least one emoji |
| `caps_ratio` | Fraction of alphabetic characters that are uppercase |
| `review_word_count` | Number of words in the review |
| `language` | ISO 639-1 language code |

**Class imbalance:** ~93% of reviews have a positive true label, ~6% neutral, ~1% negative. Use Macro-F1 as the primary metric ‚Äî accuracy alone is misleading here.

## Results Summary

| Model | Accuracy | Macro-F1 |
|---|---|---|
| VADER | 51.7% | 27.2% |
| HuggingFace RoBERTa | 36.6% | 22.3% |

*Macro-F1 is the primary metric given severe class imbalance (~93% positive reviews).*

## Key Findings

- **VADER outperforms HuggingFace on both metrics despite being rule-based.** VADER achieves 51.7% accuracy and 27.2% Macro-F1 versus HuggingFace's 36.6% accuracy and 22.3% Macro-F1. The RoBERTa model was trained on tweets, which may differ enough from Letterboxd's enthusiastic long-form review style to hurt its calibration.
- **Both models struggle badly with the minority classes.** Negative and neutral reviews make up only ~7% of the dataset, and neither model reliably identifies them ‚Äî precision on the negative class is ‚â§2% for both. The overwhelming positive majority pulls predictions toward positive, masking poor recall on underrepresented sentiments.
- **Review length has a modest effect on VADER accuracy but not a systematic one.** Neither model shows a clear monotonic improvement with longer reviews, suggesting that word count alone does not determine whether a review's textual sentiment aligns with its star rating ‚Äî ironic or ambiguous language is equally likely at any length.
